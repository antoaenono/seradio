# MVP Features

## Backend

### Library

- Accept a path to a single file
- Proof file types to match accepted files for player backend upon consumption of new files

### Scheduling

- A single song on repeat

### Streaming

- Stream to more than 1 player simultaneously

## Frontend

### Player Page

- See current playing track's metadata
- One button to: Stop streaming / resume to live
- Light mode

## Deployment

- Local
